Canvas 2D text rendering

HTML rich text,
drawn on canvas.

render-tag takes HTML + CSS and draws it onto a canvas element using pure 2D API calls. No SVG. No foreignObject. Synchronous. Zero dependencies.

By design, render-tag focuses on rich text only — paragraphs, headings, lists, tables, inline formatting. It does not try to render arbitrary HTML like form controls, iframes, or complex layouts. This focus is what makes it fast.

Performance

API

      

      
renderHTML(html, options) → { canvas, height }

      

        
          
OptionTypeDefaultDescription

        
        
          
widthnumber—Layout width in CSS pixels. Required.

          
heightnumberautoFixed height. Auto-sized from content if omitted.

          
cssstring''Additional CSS. Supports @font-face, classes, selectors.

          
canvasHTMLCanvasElementcreatedRender onto an existing canvas.

          
pixelRationumber1Device pixel ratio for HiDPI.

          
useDomMeasurementsbooleantrueDOM probes for cross-browser line height accuracy.

        
      

      
Synchronous. Fonts must be loaded before calling.

    

  


  
  

    

      

        
import { renderHTML } from 'render-tag';

const { canvas, height } = renderHTML(
  '<p>Hello <strong>world</strong></p>',
  { width: 400, pixelRatio: window.devicePixelRatio }
);

document.body.appendChild(canvas);
